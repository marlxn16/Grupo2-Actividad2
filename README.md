UNIVERSIDAD DE LAS FUERZAS ARMADAS 'ESPE' 

Grupo#2

Este proyecto consiste en una calculadora web funcional, desarrollada con HTML, CSS, JavaScript y Bootstrap 5, como parte de la práctica de aprendizaje del Grupo 2.
El objetivo fue integrar el uso de componentes de Bootstrap, el manejo del DOM con JavaScript, y el diseño responsive para lograr una interfaz moderna, limpia y funcional.


Creamos esta página usando Bootstrap 5 para demostrar cómo se pueden combinar sus componentes con la funcionalidad de JavaScript en un proyecto práctico. En la parte superior colocamos una barra de navegación con el nombre del grupo y un fondo azul, utilizando las clases prediseñadas del framework para mantener un diseño limpio y ordenado.

La sección principal fue la calculadora, donde organizamos los botones dentro de una tabla para simular el panel clásico. Cada botón fue creado con Bootstrap, aplicando colores distintos para identificar las funciones: los botones grises representan los números y operaciones básicas, el botón azul (=) muestra el resultado y el botón rojo (C) sirve para limpiar toda la pantalla. También añadimos un botón Borrar para eliminar solo el último número ingresado, y un botón para el punto decimal.

El área superior de la calculadora muestra un campo de texto con fondo oscuro y letras verdes, donde se visualizan las operaciones antes de obtener el resultado. Todo el diseño fue trabajado con CSS externo, aplicando un fondo degradado azul, bordes redondeados y transiciones suaves al pasar el cursor por los botones, para que la interfaz se vea moderna y agradable.

La parte lógica fue desarrollada con JavaScript, asignando eventos a cada botón para que al hacer clic agreguen su valor en pantalla. El botón igual evalúa la expresión completa con eval() y muestra el resultado final debajo del área de texto. Además, agregamos control de errores para mostrar el mensaje “Sintaxis inválida” cuando la operación no es válida.

Con este proyecto aprendimos a combinar Bootstrap con código JavaScript y CSS personalizado, a mantener una estructura organizada en carpetas y a crear una aplicación web completamente funcional y responsive. El resultado fue una calculadora sencilla, moderna y práctica, que refleja el uso correcto de los componentes básicos de Bootstrap junto con la lógica de programación.
